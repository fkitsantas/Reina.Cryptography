Task< byte[]> Reina.Cryptography.Interfaces.IKeyManager.GetEncryptionKeyAsync ( string  keyName)

Asynchronously retrieves a 256bit encryption key based on the provided key name.

Parameters
keyNameThe unique name of the key to retrieve.
Returns
A task that represents the asynchronous operation. The task result contains the encryption key as a byte array.
Exceptions
System.ArgumentNullExceptionThrown when the keyName is null or empty.
System.Security.Cryptography.CryptographicExceptionThrown when the key retrieval fails due to issues such as missing key, access issues, or backend service errors.
